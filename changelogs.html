V1.3

Released 3/26/22

Been Working On This All Spring Break!

  • Account database has been reset for
    security reasons
  • Your passwords are now
    securely stored using the bcrypt
    hashing algorithm (they cannot be unhashed,
    meaning your password is never stored in
    plain text

    more modern and polished

    search system, now you can search your
    pastes for any keywords

    - Share raw

    Not nearly as good as I want it
    (going to redesign later on)
  • Mobile App Not Planned Anymore

    one instead (progressive web app)
